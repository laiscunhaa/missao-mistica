Carta da Fase 1

Decifre a Última Frase

Lembrete da Cifra de César

Cada letra é deslocada 3 posições no alfabeto:

A → D
(e vice-versa)
B → E
(e vice-versa)
C → F
(e vice-versa)
... → ...
(padrão continua)

Mensagem Criptografada:

XOWLPDFRIDVH

Esta mensagem foi cifrada usando a Cifra de César com deslocamento fixo de 3 posições.

Exemplo de Decifração

Vamos decifrar as primeiras letras juntos:

X → -3 posições → U

O → -3 posições → L

W → -3 posições → T

Continue com as letras restantes para descobrir a frase completa!

Objetivo: Decifre "ULTIMAFRASE" usando Cifra de César com deslocamento fixo de 3
