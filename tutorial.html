Decifre a Mensagem

Cifra de César (Deslocamento: 3)

A Cifra de César desloca cada letra 3 posições no alfabeto:

A → D
B → E
C → F
... → ...
X → A
Y → B
Z → C

Para decifrar: Faça o processo inverso (volte 3 posições).

Mensagem Criptografada:

WXWRXLD

Esta mensagem foi cifrada usando a Cifra de César com deslocamento de 3 posições.

Dica: A palavra tem 7 letras e começa com T
