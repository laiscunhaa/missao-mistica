Decifre com Matrizes

Como funciona a Cifra de Matriz 2x2

1. Converta cada letra em número (A=0, B=1, ..., Z=25, Espaço=26)

2. Separe os números em pares [x, y]

3. Multiplique pela matriz:
[x'] = [a b] [x] mod 27
[y'] [c d] [y]

Mensagem Criptografada:

JMDASCPOGB

Esta mensagem foi cifrada usando uma matriz 2x2. A mensagem original é "AOCONTRARIO" (11 letras).

Matriz de Criptografia:

3
2
1
1

Matriz = [[3, 2], [1, 1]]

Exemplo de Cálculo

Vamos decifrar as primeiras letras "JM":

Passo 1: Converter para números
J = 9, M = 12

Passo 2: Encontrar matriz inversa de [[3, 2], [1, 1]] mod 27
Determinante = (3×1 - 2×1) = 1
Inverso do determinante = 1 (pois 1×1 mod 27 = 1)
Matriz inversa = [[1, -2], [-1, 3]] mod 27 = [[1, 25], [26, 3]]

Passo 3: Multiplicar [9, 12] pela matriz inversa
x' = (1×9 + 25×12) mod 27 = (9 + 300) mod 27 = 309 mod 27 = 15
y' = (26×9 + 3×12) mod 27 = (234 + 36) mod 27 = 270 mod 27 = 0

Passo 4: Converter números de volta para letras
15 = P, 0 = A
Resultado: "PA"

Dica: A mensagem começa com "AO"
